# Loveuinsta
This project send heart to an instagram username repeatedly.

## setup
``` 
    clone the repo
    run >> npm install
```

```
    set .env file to 
    FROM_ID ='' // from which id you want to send hearts
    FROM_ID_PASS='' // password of from id
    TO_ID ='' //to which id you want to send heartd
```

```
    run >> sh loopit.sh
```
